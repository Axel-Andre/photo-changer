# LinkedIn Photo Changer extension

Extension pour tester de changer visuellement la photo de profile d'un personne sur LinkedIn

## Installation
### Installer les modules
```
yarn install
```
### Build l'extension

``` 
yarn build
```

### Lancer en développement

``` 

yarn start
```
